À propos

01. Qui suis-je ?

Je suis un ingénieur en génie minier passionné par la simulation, la modélisation et l'analyse de données. Mon parcours combine l'expertise métier minière avec des compétences techniques en développement et en intelligence artificielle.

Dans le cadre de mon mémoire de maîtrise, j'ai développé un modèle de simulation d'opérations minières sous AnyLogic : flotte de camions, pelles, bulldozers, stations carburant, modèle de pannes (MTBF/TBF), tonnages par palier. Le modèle permet d'évaluer les performances opérationnelles et d'optimiser la planification de la production.

Récemment, j'ai construit Command Center : une plateforme combinant un dashboard web intelligent pour l'analyse de données multi-feuilles et un agent conversationnel WhatsApp propulsé par Claude (Anthropic) pour interroger les données en langage naturel.

Je cherche à appliquer cette double compétence — ingénierie métier + technologies modernes — à des projets qui ont un impact.

Command Center — Dashboard + Agent IA WhatsApp

Plateforme d'analyse de données intelligente combinant une interface web adaptative et un agent conversationnel WhatsApp propulsé par Claude (Anthropic).

  • Dashboard web single-file avec détection automatique de types (numérique, date, catégorie, spatial)
  • Visualisations adaptatives : histogrammes, camemberts, cartes X/Y, timelines, courbes cumulées
  • Support multi-feuilles Excel avec schéma et insights détectés par feuille
  • Agent WhatsApp : Claude appelle des outils (lecture Excel, agrégations, graphiques) via tool use
  • Adaptive thinking + prompt caching pour optimiser coûts et performance
  • Génération de graphiques envoyés en image MMS directement sur WhatsApp
